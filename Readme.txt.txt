Project Name: Firebase React Website Builder
Overview
This project is a web application built with TypeScript and the React framework, designed to create websites 
that leverage Google authentication for user login and Google Firebase for data storage. 
The primary focus of this application is to provide a seamless user experience for 
logging in through Google accounts and posting information using Firebase.

Features
Google Authentication
Users can log in using their Google accounts, ensuring a secure and streamlined authentication process.

Firebase Integration
Utilizes Google Firebase for data storage, providing a scalable and real-time database for storing user-generated content.

React Framework
Developed with the React framework, ensuring a modular and efficient component-based structure for easy maintenance and scalability.

TypeScript
TypeScript is used to enhance the codebase by adding static typing, making the code more robust and maintainable.
